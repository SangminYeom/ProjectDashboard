# 전체 UI Apple-style 리디자인 설계서

- 날짜: 2026-06-10
- 상태: 승인
- 범위: 홈 카드 + 프로젝트 상세 페이지 전체

## 목표

전체 앱을 Apple-style Minimal 디자인으로 통일한다.

현재 문제:
- 섹션 배경 박스, 이모지 상태 표시, 과다한 색상으로 "촌스럽고 올드하다"
- 중점수행과제가 시각적으로 강조되지 않음
- 홈 카드·상세 페이지 간 디자인 일관성 없음

## 디자인 원칙

1. **색은 하나** — 액센트 `#0071e3`(Apple Blue)만 사용. 진척률·링크에만.
2. **배경 박스 없음** — 섹션 배경색·테두리 박스 대신 여백과 레이블로 구분
3. **여백이 곧 디자인** — 넉넉한 패딩, 섹션 간 24px 간격
4. **이모지 전면 제거** — 상태는 CSS 5px dot, 아이콘은 텍스트 기호
5. **진척 바 2–4px** — 얇고 절제된 선

| 요소 | 크기 | 굵기 | 비고 |
|---|---|---|---|
| 페이지 제목 | 28–32px | 700 | letter -0.6px |

```css
background: #fff;
border-radius: 14–18px;
box-shadow: 0 2px 8px rgba(0,0,0,.04), 0 0 1px rgba(0,0,0,.06);
/* 호버 */
transform: translateY(-2px);
box-shadow: 0 8px 32px rgba(0,0,0,.08), 0 0 1px rgba(0,0,0,.06);
```

## 1. 홈 페이지 (`Home.jsx` + `styles.css`)

### 페이지 헤더

```
프로젝트 대시보드          ← 32px bold, #1d1d1f
3개 프로젝트  • 지연 2건  • 미해결 3건   ← 13px, #6e6e73, dot 구분
```

### 프로젝트 카드 구조

```
┌──────────────────────────────────┐  border-radius 18px, padding 28px 26px
│  프로젝트 이름                    │  17px 600, #1d1d1f, letter -0.4px
│  2026년 1월 – 12월               │  12px, #86868b
│                                  │
│  중점수행과제  ← 섹션 레이블       │  9.5px uppercase #86868b, mb 12px
│  과제명 ──────────────────  80%  │  name + 2px bar(#1d1d1f) + % (#86868b)
│  과제명 ──────────────────  45%  │  과제명 잘림 없음 (white-space: nowrap)
│  과제명 ──────────────────  20%  │
│                                  │  섹션 간격 24px
│  KPI  ← 섹션 레이블              │
│  KPI명          7억/10억  진척70% │  name(#6e6e73, 좌) / 수치+진척(우)
│  KPI명    3,200/5,000명  진척64%  │  진척 % → #0071e3 bold
│  KPI명                    달성   │  정성형 → #6e6e73
│                                  │
│ ─────────────────────────────── │  1px #f0f0f2
│  ● 정상 5  ● 주의 1  ● 고려 2건  │  5px dot + 11px #86868b 텍스트
└──────────────────────────────────┘
```

**규칙:**
- 섹션 배경 박스 없음
- KPI 수치형: 진척 바 없음 (텍스트 수치로 충분)
- KPI 정성형: `#6e6e73` 텍스트만

## 3. KPI 바 (`KpiBar.jsx`)

카드형 가로 스크롤 레이아웃 유지. 각 카드 스타일 변경:

```
┌──────────────────────┐
│  연간 매출 목표  ✏  ✕  │  14px 600, 버튼 우측
│                       │
│       70%             │  26px 700, #1d1d1f (수치형)
│  ████████░░  7/10억   │  4px gauge, #0071e3 fill
│                       │
│  (정성형) 순항        │  select 유지, 이모지 옵션 텍스트만으로 변경
└──────────────────────┘
```

- 카드: border-radius 14px, 패딩 16px 18px
- gauge: 4px, fill `#0071e3`, track `#e8e8ed`
- 정성형 select: 이모지 제거 → `달성 / 순항 / 주의 / 미달`

## 4. 중점수행과제 (`Initiatives.jsx`)

### 과제 카드

```
┌────────────────────────────────────┐  border-radius 14px
│ ⠿  인프라 전환 프로젝트   ✏  ✕    │  drag handle(#d2d2d7) + 제목 + 버튼
│     진척 48% · 태스크 3건 · 담당 홍길동  │  13px, #86868b
└────────────────────────────────────┘
  [펼쳐진 Gantt]
```

- 카드 배경: `#fff`, 박스섀도우 공통 스타일
- 헤더 `align-items: center`, gap 10px
- drag-over 상태: `outline: 2px solid #0071e3`

### Gantt 행

```
⠿  레거시 데이터 마이그레이션   [50%]  ████░░░░░░░  예정   ✏  ✕
```

- 날짜 레이블(헤더 행): 9.5px uppercase, `#86868b`
- 진척 바: 4px, fill `#1d1d1f`, track `#e8e8ed`
- 지연 바: fill `#ff3b30`
- 완료 바: fill `#30d158`
- 상태 텍스트: `#86868b` / 지연 → `#ff3b30`
- drag-over 행: background `#f0f5ff`

## 5. 운영업무 (`OperationsTable.jsx`)

테이블을 흰 카드로 감싸고 내부 스타일 정리:

```
┌──────────────────────────────────────────────────────┐
│  업무    주기    담당    상태      메모           [삭]  │  헤더: 9.5px uppercase #86868b
│ ─────────────────────────────────────────────────── │
│  주간 보고  주  홍길동  ● 정상  입력 가능한 메모  ✕   │  상태: dot(5px) + 텍스트
│  월간 결산  월  김팀장  ● 주의  검토 필요        ✕   │  select 이모지 제거
└──────────────────────────────────────────────────────┘
```

- 테이블 wrapper: 흰 카드 (border-radius 14px, 패딩 없이 overflow: hidden)
- `th`: 9.5px uppercase, `#86868b`, padding 12px 16px
- `td`: 13px, padding 10px 16px
- 행 구분선: 1px `#f0f0f2`
- 상태 select: 이모지 제거 → dot으로 시각화, `정상 / 주의 / 이슈`
